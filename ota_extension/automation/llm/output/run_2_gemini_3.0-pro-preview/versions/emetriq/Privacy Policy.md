4.  _Websites klassifizieren_

Der RTB ist ein automatisiertes Verfahren, welches im Bruchteil von Sekunden zwischen verschiedenen Bietern für eine Werbeanzeige denjenigen auswählt, der das passende Gebot anbietet.
